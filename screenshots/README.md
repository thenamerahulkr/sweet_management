# Screenshots

This directory contains screenshots of the Sweet Shop Management System application.

## Required Screenshots:

1. **login.png** - Login page interface
2. **register.png** - Registration page with role selection
3. **dashboard.png** - Main dashboard with sweet grid
4. **search.png** - Search and filter functionality
5. **admin.png** - Admin features (add, edit, delete sweets)
6. **purchase.png** - Purchase flow interface

## How to Add Screenshots:

1. Run the application locally
2. Take screenshots of each major feature
3. Save them in this directory with the names listed above
4. Update the README.md if needed

## Screenshot Guidelines:

- Use high-quality images (PNG format preferred)
- Show the application in a clean browser window
- Demonstrate key features and functionality
- Include both desktop and mobile views if possible
